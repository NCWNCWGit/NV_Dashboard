Quick Start Guide

  1. View a Dashboard: Click any dashboard card above
  2. Add Data: Place CSV files in the data/ folder
  3. Create New Dashboard: Copy dashboards/template/ to dashboards/[name]/ and customize
  4. Edit Dashboards: Use NPL or any text editor to modify HTML files
  5. Development Server: Run python -m http.server 8000 for better testing

Project Structure

  • dashboards/ - All dashboard pages
  • shared/ - CSS, JS utilities, and libraries
  • data/ - CSV data files
  • README.md - Full documentation
